# Examples

This directory contains sanitized, documentation-oriented examples that illustrate the shape of Debrief runtime artifacts without exposing the private implementation.

Included examples:

- `sample-run-state.md`: durable runtime state and resume boundary shape
- `sample-canonical-summary.md`: authoritative reviewer-facing artifact summary
- `sample-artifact-lineage.md`: upstream inputs, workflow transitions, and publication state

Examples here are intended to support evaluation of the runtime model, not to recreate the full operational system.
